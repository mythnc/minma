# Typescript Development Test

## Development Environment

- npm 6.14.4
- Node.js 12.16.2
- TypeScript 3.8.3
- gulp-cli 2.2.0 (global)
- Gulp 4.0.0
- jest 25.3.0

## How-to

### Install
```bash
$ npm install
```

### Compile & Run
```bash
$ gulp
$ node dist/main.js
```

### Testing
```bash
$ npx jest
```
